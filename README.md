# GenInvoice (MERN) + Tauri Desktop

This project runs:

- **Frontend**: Vite + React
- **Backend API**: Express (in `Backend/`) on `SERVER_PORT` (default `5002`)
- **Desktop app**: Tauri (in `src-tauri/`)

## Environment variables

- **Frontend** (`.env`):
  - `VITE_API_URL=http://localhost:5002`
- **Backend** (`Backend/.env`):
  - `SERVER_PORT=5002`
  - `MONGODB_URI=...`

## Run (web dev)

```bash
npm install
npm run app
```

This starts:
- Vite on `http://localhost:5173`
- Backend on `http://localhost:5002`

## Run (desktop dev with Tauri)

### Prerequisites (Windows)

- Install **Rust** (includes `cargo`): `rustup` toolchain
- Install **Microsoft C++ Build Tools** (MSVC)
- Ensure **WebView2 Runtime** is available (usually already on Windows 10/11)

Then:

```bash
npm run tauri:dev
```

Note: Tauri dev runs `npm run app` automatically (starts Vite + backend).

## Build Windows installer

```bash
npm run tauri:build
```

The output installer will be generated by Tauri (NSIS target).
